# HSBC Customer Churn Analysis

## 📊 Project Overview
This project analyzes customer churn using machine learning and data visualization.

## 🔧 Tools Used
- Python
- Pandas
- Seaborn
- Scikit-learn

## 📈 Key Insights
- Age is the most important factor affecting churn
- Inactive customers are more likely to leave
- Customers with fewer products show higher churn

## 🤖 Model
- Random Forest Classifier
- Accuracy: ~87%

## 📊 Visualizations
- Correlation Matrix
- Feature Importance
- Churn Distribution
- Age vs Churn
- Active Member vs Churn

## 🚀 Conclusion
Data-driven strategies can help reduce customer churn and improve retention.
